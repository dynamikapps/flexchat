FlexChat Embeddable Widget Demo

This page demonstrates how to embed the FlexChat widget on any website using a simple script tag.

How to Embed FlexChat

Add the following code to your website:

<!-- FlexChat Widget -->
<script>
  // FlexChat Configuration
  window.flexChatConfig = {
    webhookUrl: "https://your-n8n-instance.com/webhook/path",
    primaryColor: "#3b82f6",
    welcomeText: "Hello! How can I help you today?",
    position: "right", // or 'left'
    logoUrl: "https://your-site.com/logo.png" // optional
  };
</script>
<script src="https://cdn.jsdelivr.net/npm/flexchat@latest/dist/flexchat.min.js" async></script>

Configuration Options

You can customize the chat widget by modifying the window.flexChatConfig object:

  • webhookUrl (required): URL to your n8n webhook
  • primaryColor: Primary color for buttons and accents
  • secondaryColor: Secondary color for highlights
  • position: 'left' or 'right' side of the screen
  • initiallyOpen: Whether the chat should be open on load
  • welcomeText: Initial message from the bot
  • responseTimeText: Text indicating expected response time
  • suggestedQuestions: Array of clickable suggested questions
  • userFields: Fields to collect from users (name, email, etc.)
